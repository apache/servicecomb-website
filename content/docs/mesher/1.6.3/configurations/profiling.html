Profile Mesher¶

Mesher has a convenience way to enable go pprof, so that you can easily analyze the performance of mesher

Configurations¶

pprof:
  enable: true
  listen: 127.0.0.0.1:6060

enable

(optional, bool) default is false

listen

(optional, string) Listen IP and port
